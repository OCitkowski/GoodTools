1С:Предприятие 8
Конфигурация "Библиотека стандартных подсистем", редакция 3.0

Библиотека включает в себя набор универсальных функциональных подсистем, 
предназначенных для использования в прикладных решениях
на платформе "1С:Предприятие 8.3" 
================================================================================

1. Файлы поставки.
2. Важная информация.

Файлы поставки
--------------------------------------------------------------------------------

1. 1Cv8.cf - файл поставки библиотеки. Не предназначен для создания 
  информационных баз по шаблону. Перед использованием необходимо 
  ознакомиться с документацией на ИТС (http://its.1c.ru/db/bspdoc).
2. 1Cv8_international.cf - файл поставки международной версии библиотеки.
  Не предназначен для создания информационных баз по шаблону.
  Перед использованием необходимо ознакомиться с документацией
  на ИТС (http://its.1c.ru/db/bspdoc).
3. 1Cv8_demo.dt - демонстрационная база.
4. 1Cv8_demo.cf - файл поставки демонстрационной базы.

5. Каталог ExtFiles:
 5.1 Подкаталог Docs:
  5.1.1 UpdateSSL.htm - описание изменений по сравнению в этой версии.
  
 5.2 Подкаталог "Синхронизация данных"
  5.2.1 Подкаталог "БСП, версия 2.2.5"
   5.2.1.1 1Cv8.dt - Информационная база версии 2.2.5 для демонстрации обратной
                     совместимости правил обмена между конфигурациями старшей и 
                     младшей версий.
  
 5.3 Особенности конфигурации (АПК).xml - файл с особенностями для загрузки
     в АПК перед проверкой конфигурации.
 5.4 ЛокализуемыеОбъектыБСП.txt - содержит список объектов библиотеки, содержащих
   российскую специфику и отсутствующих в файле поставки международной библиотеки.

Файлы расположены в подкаталоге

\1c\SSL\3_0_1_305\

каталога шаблонов 1С:Предприятия 8.3.


Важная информация
--------------------------------------------------------------------------------
Данная версия конфигурации предназначена для использования 
с платформой "1С:Предприятие 8.3" не ниже версии 8.3.12.1412 с отключенным
режимом совместимости.
